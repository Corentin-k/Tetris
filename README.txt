Jeu "Tetris"

Réalisé par Corentin KERVAGORET et Jonathan REN en Décembre 2022

Projet réalisé dans le cadre du cours de L1 "Programmation en Python" à EFREI

Le projet contient :
	-5 fichiers python .py (main, Bloc, Fonction, Fonction_Affichage,Fonction_Saisie)
	-3 fichiers fichiers texte .txt (Cercle, Losange, Triangle)

Pour lancer le jeu, il vous faut disposé d'une interface python (ex: PyCharm)
Et exécuter le fichier main.py.
Une fois ouvert, appuyez sur le bouton play (triangle vert) en haut à droite (sur PyCharm)
Enfin, suivez les consignes dans la console.

Toutes les actions se déroulent dans la console. 
Vous n'aurez donc besoin que de votre clavier.
Lorsque que l'on vous demande quelque chose ne répondez que le stricte minimum ne mettez pas d'espace et faites attention à bien mettre en majuscule lorsque necessaire.

Il n'y a aucun bug connu à ce jour. Si vous trouvez un quelquonque bug veuillez contactez les créateurs du jeu.

Ce jeu est en béta, de nombreuses améliorations sont à prévoir, si vous voulez proposez des modifications vous pouvez contactez les créateurs du jeu.
Il est par exemple prévu de réaliser une sauvegarde du score pour avoir un tableau de scores.

Bon jeu....
